Interface Modules

Interface modules provide access to hardware on or connected to the platform that MQTTany is running on.

GPIO

Allows you to control GPIO pins on single board computers.

I2C

Provides access to devices over I2C using the Linux kernel drivers.

OneWire

Provides access to Dallas OneWire devices using the w1 kernel module.

LED

Provides control of WS281x and similar LEDs through animation scripts.

XSET

Allows xset commands to be executed to set screensaver and display power options.
